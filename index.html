Projects

Longjohn
Longjohn is a software to manage studios and equipment for the sound engineers from the UDK Berlin.
At the moment only the equipment reservation works, but there is at least some code to look at.
Healthety
We developed Healthety at Käuferportal because we wanted to monitor the load from every server in realtime.
And now we can use this tool on nearly any other number as well. It's so simple to add a chart that we now have like 10 different reporters on about 20 machines reporting their numbers (load, memory, revenue, mail delivery, outgoing calls, db replica state, …).
Soldat.js
There is a big vision behind Soldat.js, we want to port the original Soldat to webgl. At the moment this repo is a playground to create a basic game and explore the possibilities. Besides from our main goal, we decided to use coffeescript just because we can.
